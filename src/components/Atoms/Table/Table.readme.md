## How to use the Table

The Table consists out of a HTML <table>, with a table head <th> or table body <td> which has rows <tr> and finally cells <td>.

```html
<table>
  <TableHead>
    <TableRow>
      <TableCell></TableCell>
      <TableCell></TableCell>
    </TableRow>
  </TableHead>
  <TableBody>
    <TableRow>
      <TableCell></TableCell>
      <TableCell></TableCell>
    </TableRow>
  </TableBody>
</table>
```
